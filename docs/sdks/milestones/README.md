## listMilestones

Retrieve a paginated list of GitHub milestones.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="list_milestones" method="get" path="/api/v0/milestones/" -->

  apiKey: process.env["NEST_API_KEY"] ?? "",
